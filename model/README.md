## Run Locally

Go to the project directory

```bash
  cd model
```

Download Yolo-v4 weights through the following command:
```bash
wget https://github.com/AlexeyAB/darknet/releases/download/darknet_yolo_v3_optimal/yolov4.weights
```

Run any file

```bash
  python main.py
```
