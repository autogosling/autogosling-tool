## Run Locally

Clone the project

```bash
  git clone https://github.com/autogosling/autogosling-tool
```

Go to the project directory

```bash
  cd autogosling-tool
```

Create a conda environment using the following command:

```bash
  conda env create -f myenv.yml
  conda activate autogosling
```

Run any file

```bash
  python main.py
```
